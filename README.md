# nisasim
つみたてNISAのシミュレーションを行うCLIです。

## 使い方

```bash
python3 nisasim.py <毎月の積立金額(円)> <予定利回り(年率%)> <運用年数>
```

元本、運用収益、将来の運用資産額を表示します。


### 実行例

```bash
python3 nisasim.py 30000 5 20
```

出力例:

```text
7200000 5131010 12331010
```
